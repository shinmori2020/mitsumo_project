### 3. Step1「サイトの基本情報」画面の実装
- **データ・ユーティリティ**
  - `src/data/pricing.js` — 全43項目の単価データ（skills/pricing/SKILL.mdと完全一致）
  - `src/utils/calculatePrice.js` — 金額計算ロジック（全ステップ対応：小計→納期倍率→割引→消費税）
  - `src/utils/formatCurrency.js` — 3桁カンマ区切りフォーマット（Math.round + toLocaleString）
  - `src/hooks/useEstimate.js` — 見積もり状態管理カスタムフック（useState + calculatePrice）
- **UIコンポーネント（6個）**
  - `PillButton` — ピル型選択ボタン（ON: グリーン背景白文字 / OFF: パステル背景）
  - `ToggleSwitch` — トグルスイッチ（項目名+補足テキスト+トグル行）
  - `Slider` — スライダー+数値バッジ（range input + バッジ表示）
  - `GroupCard` — グループラベル（グリーン背景）+ボックス
  - `StepIndicator` — 上部ステップインジケーター（5ステップ丸+ライン）
  - `PriceBar` — 下部金額バー（税抜+税込表示、ダークグリーン背景）
- **Step1画面の入力項目**
  - サイトの種類（コーポレート/LP/EC/ブログ）— ピル型選択
  - 制作方式（WordPress/HTML・CSS）— ピル型選択、排他
  - トップページ — トグル
  - 下層ページ数（0〜30）— スライダー
  - LPページ数（0〜10）— スライダー
  - テンプレート3種（ブログ記事/アーカイブ/404）— トグル、補足テキスト付き

  - データ移行 — トグル

- **ブレークポイント設定**
  - 1024px以下（タブレット）: サイドバー非表示、1カラムレイアウト、切り替えスイッチ非表示
  - 768px以下（スマホ）: パディング・フォント縮小、トグル行/スライダー行コンパクト化
  - 480px以下（小画面）: スライダー折り返し、PriceBar縦並び、ステップ間ライン短縮

  6. ボタンの位置を `top: 16px; right: 16px` → `top: 20px; right: 24px` に調整（ヘッダーpaddingと統一）

### 12. Step4「その他オプション」画面の実装
- 最適化・設定グループ（トグル×6）: SSL/SEO/速度最適化/クロスブラウザ/GA/ファビコン
- 納品関連グループ（トグル×4）: 画像書き出し/テスト検証/マニュアル/修正対応
- 保守プラン（ドロップダウン: なし/ライト月額5,000円/スタンダード月額15,000円）
- 納期（ドロップダウン: 通常×1.0/急ぎ×1.3/特急×1.5）— 補足テキスト付き
- 割引（ピル選択: なし/値引き額/割引率 + 条件に応じた数値入力表示切替）
- 見積もりヘッダー情報（お客様名・納期の希望）
- 備考情報（デザイン修正回数・支払いタイミング・支払い方法・その他備考）
- ボタンは「戻る」と「結果を見る」（通常の「次へ進む」ではない）

### 13. Step5「見積もり結果」画面の実装
- **合計金額カード**: ダークグリーン背景、お客様名表示、税込金額40px、区切り線、税抜/消費税/割引表示
- **保守プラン**: 「なし」以外の場合に月額料金を別途表示
- **有効期限**: 見積日〜30日後を自動計算・表示
- **内訳テーブル**: トグルで表示/非表示切替、ON項目のみ行生成、偶数行パステル背景
- **集計行**: 小計(税抜)/納期調整(該当時のみ)/割引(該当時のみ)/消費税(10%)/合計(税込)
- **出力ボタン**: 2×2グリッド（Excel/PDF/テキストコピー/メール）各色アイコン付き
- **下部ボタン**: 「内容を修正する」→Step4へ戻る / 「新しい見積もりを作成」→確認ダイアログ後リセット
- Step5ではPriceBar（下部金額バー）を非表示に
- `src/utils/buildBreakdown.js` — 内訳テーブル用データ生成ヘルパー新規作成

### 14. 出力機能の実装（4種）

#### Excel出力
- 当初 `xlsx`（SheetJS）で実装 → スタイルなしで見づらいため `xlsx-js-style`（MITライセンス・無料フォーク）に差し替え
- **3シート構成**:
  - Sheet1「見積もり明細」: タイトル(18px結合)、ヘッダー情報、合計金額(ダークグリーン背景結合セル)、明細テーブル(グリーンヘッダー/偶数行パステル/罫線)、集計行(太字/合計行ダークグリーン白文字)
  - Sheet2「サイト仕様」: 基本情報/デザイン設定/機能一覧/参考情報（セクションヘッダーグリーン背景、ラベルセルパステル背景）
  - Sheet3「備考・補足事項」: 注意事項/保守サポート/支払い条件（同様のスタイル）
- ファイル名: `MitsuMO_見積書_[客先名]_[YYYYMMDD].xlsx`

#### PDF出力
- `html2canvas` + `jsPDF` 方式（日本語フォント問題を回避するためブラウザレンダリング方式を採用）
- HTML テンプレートを動的に生成 → 非表示のDOMに挿入 → html2canvasでキャプチャ → jsPDFに変換
- **複数ページ分割修正**: 当初の実装でページ先頭が途切れる問題 → ピクセルとmmの変換ロジックを修正し正確なスライス処理に
- ファイル名: `MitsuMO_見積書_[客先名]_[YYYYMMDD].pdf`

#### テキストコピー
- `navigator.clipboard.writeText()` でクリップボードにコピー
- フォールバック: `document.execCommand('copy')` 対応
- 罫線風テキスト装飾（━━ / ── 等）で見やすく整形

#### メール送信
- `mailto:` リンクで件名・本文を自動挿入
- 件名: `【MitsuMO】お見積書 - ○○ 様`

### 15. UX改善（3項目）
- **ステップ切り替え時のスクロールリセット**: `handleNext` / `handleBack` で `window.scrollTo({ top: 0, behavior: 'smooth' })` を実行
- **PDF出力時のローディング表示**: ボタンテキストが「PDF生成中...」に変化、`disabled` で連打防止、生成完了後に自動復帰
- **完了済みステップのクリック遷移**: StepIndicatorの完了済み（✓マーク）ステップをクリックでそのステップに直接遷移、ホバーで `scale(1.1)` エフェクト、未到達ステップはクリック不可

### 16. その他の修正
- 内訳テーブルの集計行ボーダーを `2px solid #1D9E75` → `0.5px solid #ccc` に変更（視覚的に軽く）
