## テキストコピー機能

### 実装するコンポーネント

`src/components/export/TextCopy.jsx`

### 動作

1. ONにした全項目の明細をプレーンテキストに変換
2. クリップボードAPIでコピー
3. コピー完了時にフィードバック表示

### テキストフォーマット

```
========================================
お見積書
========================================

[客先名] 様

見積日：[YYYY年MM月DD日]
見積番号：EST-[YYYY]-[0001]
有効期限：[YYYY年MM月DD日]

----------------------------------------
【明細】
----------------------------------------
1. WordPress構築基本費        ×1   80,000円
2. トップページ コーディング    ×1   50,000円
3. 下層ページ コーディング      ×5  150,000円
4. お問い合わせフォーム         ×1   30,000円
...

----------------------------------------
小計（税抜）                     430,000円
消費税（10%）                     43,000円
========================================
合計（税込）                     473,000円
========================================

※ 見積もり有効期限：30日間
```

### クリップボードAPI

```javascript
const handleCopy = async (text) => {
  try {
    await navigator.clipboard.writeText(text);
    // フィードバック表示（例：「コピーしました」を2秒間表示）
  } catch (err) {
    // フォールバック：textarea経由でコピー
  }
};
```

### フィードバック

- ボタンのテキストを一時的に「コピーしました！」に変更
- 2秒後に元のテキスト「テキストコピー」に戻す
- ボタンの背景色も一時的にメインカラー（#1D9E75）に変更

## メール送信機能

### 実装するコンポーネント

`src/components/export/MailSend.jsx`

### 動作

mailto:リンクで実装。クリック時にデフォルトのメールクライアントを起動。

### メール内容

```
件名：【お見積書】[客先名]様 WEBサイト制作

本文：
[客先名] 様

お世話になっております。
WEBサイト制作のお見積りをお送りいたします。

━━━━━━━━━━━━━━━━━━
お見積り金額
━━━━━━━━━━━━━━━━━━
税抜：[金額]円
税込：[金額]円

【主な項目】
・[制作方式]（[基本費用]円）
・ページ制作（[ページ数]ページ）
・[主要機能1]
・[主要機能2]
...

見積もり有効期限：[有効期限日]まで

詳細な明細につきましては、
別途Excel/PDFファイルにてお送りいたします。

ご不明な点がございましたら、
お気軽にお問い合わせください。

よろしくお願いいたします。
```

### mailto:リンクの生成

```javascript
const subject = encodeURIComponent(`【お見積書】${clientName}様 WEBサイト制作`);
const body = encodeURIComponent(generateEmailBody(estimateData));
const mailtoLink = `mailto:?subject=${subject}&body=${body}`;
window.location.href = mailtoLink;
```

### 注意事項

- mailto:のbodyには文字数制限がある（ブラウザ依存、約2000文字目安）
- 長すぎる場合は主要項目のみに絞り、「詳細はExcel/PDFをご確認ください」で省略
- 改行は %0D%0A でエンコード
