# Step5「見積もり結果」実装指示

このコマンドを実行する前に、必ず以下を参照すること：
- skills/step-screens/SKILL.md（Step5の画面仕様）
- skills/pricing/SKILL.md（金額計算フロー）
- skills/design-system/SKILL.md（デザイン仕様）

---

## 概要

Step5は全入力の集約・表示・出力画面。金額バー（下部固定）はこの画面では非表示にし、代わりに結果カードで金額を表示する。

## 実装するコンポーネント

`src/components/steps/Step5Result.jsx`

## 実装内容

### 1. 合計金額カード

- 背景：#085041、border-radius: 20px、padding: 28px 24px
- 中央配置、テキストも中央
- 上部：「お見積り金額」ラベル（14px、#9FE1CB）
- お客様名（入力されていれば「○○ 様」と表示、16px、#5DCAA5）
- 税込金額（40px、font-weight: 500、#FFFFFF）＋「円」（18px、#9FE1CB）
- 区切り線（border-top 0.5px solid #0F6E56）
- 税抜金額と消費税を横並び（18px、#5DCAA5）
- 割引がある場合は「割引 −○○円」も表示

### 2. 内訳テーブル

- トグルスイッチで表示/非表示を切り替え
- デフォルト：表示状態
- テーブル構成：
  - ヘッダー行：項目 / 数量 / 単価 / 小計（背景#1D9E75、白文字）
  - ONにした項目のみ行を生成
  - 偶数行に#E1F5EE背景
  - 集計行（border-top 2px solid #1D9E75）：
    - 小計（税抜）
    - 納期調整（通常以外の場合のみ。例：「急ぎ ×1.3」）
    - 割引（ある場合のみ。例：「値引き −30,000円」or「10%割引 −43,000円」）
    - 消費税（10%）
    - 合計（税込）— 15px、#085041で強調

### 3. 出力ボタン（2×2グリッド）

- Excel出力 / PDF出力 / テキストコピー / メール送信
- 各ボタンにアイコン（色付き四角）とラベル
- クリック時の動作は各export系コマンドを参照

### 4. 下部ボタン

- 「内容を修正する」：Step4に戻る（全入力データは保持したまま）
- 「新しい見積もりを作成」：確認ダイアログ表示後、全データリセット→Step1へ

## 保守プランの表示

保守プランが「なし」以外の場合、合計金額カードの下に別途表示：
- 「※ 別途、月額保守料金：○○円/月」

## 有効期限の表示

合計金額カードの下、または内訳テーブルの下に：
- 「見積もり有効期限：[見積日] 〜 [見積日+30日]」
