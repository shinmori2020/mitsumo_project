# Excel出力仕様

SheetJSを使用した3シート構成のExcel出力の詳細仕様。

---

## 技術仕様

- ライブラリ：SheetJS（xlsx）
- ブラウザ内で完結（サーバー不要）
- ファイル名：`MitsuMO_見積書_[客先名]_[YYYYMMDD].xlsx`

---

## Sheet1：見積もり明細

### ヘッダーエリア（上部）

| セル位置 | 内容 |
|----------|------|
| 中央 | 「お 見 積 書」（タイトル、大きめフォント） |
| 左側 | [客先名] 御中 |
| 右側 | 見積日：[YYYY年MM月DD日] |
| 右側 | 有効期限：[見積日 + 有効期限日数] |
| 右側 | 見積番号：EST-[YYYY]-[連番4桁] |
| 右側 | 納期：[入力された納期希望] |

### 合計金額エリア

- 「お見積り合計金額（税込）」ラベル + 合計金額を大きく表示

### 明細テーブル

| 列 | 内容 |
|----|------|
| No. | 連番 |
| 項目 | 項目名 |
| 単価 | 円書式（#,##0"円"） |
| 数量 | 数値 |
| 小計 | 単価×数量（円書式） |
| 備考 | 項目の補足説明 |

- ONにした項目のみ行を生成（OFFの項目は含めない）
- ページ数系は数量にページ数を入れる
- 偶数行にアクセント背景色

### 集計エリア

| 行 | 内容 |
|----|------|
| 小計（税抜） | 明細の小計合計 |
| 納期調整 | 「急ぎ」「特急」の場合のみ表示（倍率と加算額） |
| 割引 | 割引がある場合のみ表示（値引き額 or 割引率） |
| 消費税（10%） | 割引後小計 × 10% |
| 合計（税込） | 割引後小計 + 消費税 |

## 書式設定

- フォント：Arial
- ヘッダー行：背景色付き（濃い色）、白文字、中央揃え
- セクションラベル：背景色付き（薄い色）、太字
- 金額：右揃え、#,##0"円" 書式
- 偶数行：薄いアクセント背景色
- 罫線：薄い罫線（thin、#CCCCCC）
- 列幅：自動調整
- 印刷設定：A4縦、ページ内に収める
