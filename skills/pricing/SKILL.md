# 単価データ

MitsuMOアプリで使用する全43項目の単価データ。
このファイルがアプリ内の金額計算の唯一の正（Single Source of Truth）である。
src/data/pricing.js に定数として定義する際は、このファイルの値と完全一致させること。

## 消費税・割引・有効期限

| 項目 | 値 |
|------|------|
| 消費税率 | 10%（TAX_RATE = 0.1） |
| 表示 | 税抜・税込の両方を常に併記 |
| 割引方式 | 値引き額（円）と割引率（%）の両方を選択可能 |
| 割引適用 | 小計（税抜）から差し引いた後に消費税を計算 |
| 有効期限 | デフォルト30日（見積もりごとに変更可能） |

## 金額計算フロー

1. 選択された項目の単価を合算 → 小計（税抜）
2. 納期倍率を適用 → 小計（税抜）× 納期倍率
3. 割引を適用 → 小計 − 値引き額 or 小計 ×（1 − 割引率）
4. 消費税を計算 → 割引後小計 × 0.1
5. 合計（税込）= 割引後小計 + 消費税
