# 画面構成

全5画面のステップウィザード。各画面の入力項目・UIコントロール・金額計算ロジックの詳細。
単価は skills/pricing/SKILL.md を参照。デザインは skills/design-system/SKILL.md を参照。

---

## Step1：サイトの基本情報

対応カテゴリ：カテゴリ1（基本費用）・カテゴリ2（ページコーディング）

### 入力項目

| 項目 | UIコントロール | 選択肢 | 金額への影響 |
|------|---------------|--------|-------------|
| サイトの種類 | ピル型選択 | コーポレート / LP / EC / ブログ | EC選択時にStep2でEC関連グループを表示 |
| 制作方式 | ピル型選択 | WordPress / HTML・CSS | WordPress → base_wp(80,000) / HTML → base_html(30,000) |
| トップページ | トグル | ON / OFF | ON → page_top(50,000) |
| 下層ページ数 | スライダー＋数値バッジ | 0〜30 | page_sub(15,000) × ページ数 |
| LPページ数 | スライダー＋数値バッジ | 0〜10 | page_lp(50,000) × ページ数 |
| ブログ記事テンプレート | トグル | ON / OFF | ON → tpl_blog(20,000) |
| アーカイブテンプレート | トグル | ON / OFF | ON → tpl_archive(15,000) |
| 404ページ | トグル | ON / OFF | ON → page_404(5,000) |
| レスポンシブ対応 | トグル | ON / OFF | ON → opt_responsive(30,000) |
| 既存サイトからのデータ移行 | トグル | ON / OFF | ON → del_migration(50,000) |

### 補足説明テキスト

- ブログ記事テンプレート：「投稿詳細の共通レイアウト」
- アーカイブテンプレート：「投稿一覧・カテゴリー一覧」
- 404ページ：「エラーページのデザイン」
- レスポンシブ対応：「スマホ・タブレット対応」
- データ移行：「リニューアル案件の場合」

### ボタン

- 「戻る」：disabled（Step1なので）
- 「次へ進む」

## Step3：デザイン仕様

金額に影響しない情報入力画面。入力データはExcel出力のSheet2に反映される。

### 入力項目

| 項目 | UIコントロール | 詳細 |
|------|---------------|------|
| 配色テーマ | カラースウォッチカード×3 | メイン・サブ・アクセントの3色。タップでカラーピッカー表示。カラーコード表示付き |
| フォントの雰囲気 | ピル型選択 | やわらかい / シャープ / フォーマル / カジュアル |
| レイアウトイメージ | 2×2グリッド画像選択 | パターンA（ヘッダー大・写真メイン）/ パターンB（サイドバー付・情報整理型）/ パターンC（1カラム・スクロール型）/ パターンD（グリッド・ギャラリー型） |
| 参考サイトURL | テキスト入力×2 | placeholder: 「https://example.com」「2つ目の参考サイト（任意）」 |
| その他の要望 | テキストエリア | placeholder: 「デザインに関する補足やイメージがあれば自由にご記入ください」 |

### ボタン

- 「戻る」
- 「次へ進む」

## Step5：見積もり結果

全入力の集約・表示・出力画面。

### 合計金額カード

- ダークグリーン背景（#085041）、中央配置
- 「お見積り金額」ラベル
- お客様名（入力されていれば表示）
- 税込金額を大きく表示（40px）
- 区切り線
- 税抜金額と消費税を並べて表示
- 割引がある場合は割引額も表示

### 内訳テーブル

- トグルで表示/非表示を切り替え（デフォルト：表示）
- 列構成：項目 / 数量 / 単価 / 小計
- ONにした項目のみ表示
- 偶数行にパステルグリーン背景
- 最下部に集計行：
  - 小計（税抜）
  - 納期調整（該当時のみ）
  - 割引（該当時のみ）
  - 消費税（10%）
  - 合計（税込）

### 出力ボタン（2×2グリッド）

| ボタン | アイコン | 動作 |
|--------|---------|------|
| Excel出力 | XL（#0F6E56） | SheetJSで3シートExcelを生成・ダウンロード |
| PDF出力 | PDF（#C00000） | jsPDF等でPDFを生成・ダウンロード |
| テキストコピー | Cp（#085041） | 明細をテキスト化しクリップボードにコピー |
| メール送信 | Mail（#2B4C7E） | mailto:リンクで本文に見積もり概要を挿入 |

### 下部ボタン

- 「内容を修正する」：Step4に戻る（入力データは保持）
- 「新しい見積もりを作成」：全入力をリセットしてStep1に戻る
