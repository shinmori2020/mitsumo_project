# アプリ基本仕様

MitsuMOアプリの基本動作仕様。

---

## 入力UI

- 方式：ステップウィザード型（5ステップ、1画面ずつ進む）
- ナビゲーション：ステップインジケーターを上部に固定表示
- 進行：「次へ進む」「戻る」ボタンで画面遷移
- 各ステップ間のデータは状態（state）として保持

## 金額表示

- 画面下部に常に合計金額を固定表示
- 税抜・税込の両方を併記
- 入力変更時にリアルタイムで再計算・表示更新

## 料金体系

- コード内に固定値で埋め込み
- 単価データは skills/pricing/SKILL.md が唯一の正
- src/data/pricing.js に定数として定義

## 消費税

- 税率：10%（定数 TAX_RATE = 0.1）
- 税抜・税込の両方を常に併記
- 小数点以下は四捨五入

## 内訳表示

- Step5（結果画面）で切り替え可能
- トグルスイッチ：合計のみ ↔ 全項目の内訳表示
- 内訳テーブル：項目 / 数量 / 単価 / 小計

## 割引機能

- 値引き額（円）の直接入力 と 割引率（%）の入力を両方選べる
- 合計金額から差し引く形式
- 割引適用後に消費税を計算
- Step4で入力、Step5で反映

## 見積もり有効期限

- デフォルト：30日
- 見積もりごとに変更可能

## 備考シートの入力方式

以下は固定値ではなく、案件ごとにテキスト入力する：
- デザイン修正の無料回数
- 支払いタイミング
- 支払い方法
- その他備考（自由記述）

## 出力機能（4種類）

### 1. Excel出力
- ライブラリ：SheetJS
- 3シート構成（詳細は skills/excel-spec/SKILL.md を参照）
- ブラウザ内で生成、ダウンロード

### 2. PDF出力
- ライブラリ：jsPDF等のブラウザ内PDF生成ライブラリ
- ロゴ：なし
- 内容：見積もり明細＋サイト仕様＋備考をシンプルに出力
- 日本語フォント対応が必要

### 3. テキストコピー
- 見積もり明細をプレーンテキストに変換
- クリップボードAPIを使用
- コピー完了時にフィードバック表示（「コピーしました」等）

### 4. メール送信
- mailto:リンクで実装（サーバー不要）
- 件名：「【お見積書】○○様 WEBサイト制作」
- 本文：仮の文面で作成（後から調整可能）
  - 挨拶文
  - 合計金額（税抜・税込）
  - 主要項目の概要
  - 「詳細はExcel/PDFファイルをご確認ください」

## 条件表示ロジック

- 制作方式「WordPress」選択時 → カテゴリ4（WordPress固有）を表示
- 制作方式「HTML/CSS」選択時 → カテゴリ4を非表示
- サイト種別「EC」選択時 → カテゴリ5（EC関連）を表示
- サイト種別「EC」以外 → カテゴリ5を非表示しグレーアウトで案内文表示
